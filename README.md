# RNLearning

Este repositório foi criado com o objetivo de aprender a desenvolver uma aplicação mobile, o objetivo principal deste aprendizado é fornecer uma base para o desenvolvimento do Trabalho de Conclusão de Curso. 

# Tecnologias Utilizadas
Este projeto foi desenvolvido utilizando as seguintes tecnologias:

Linguagem de Programação: 
TypeScript
Framework Mobile: 
React Native
Expo (para simplificar o desenvolvimento e testes)
Ferramentas de Gerenciamento de Pacotes: 
npm

# Instrução de Instalação

1. Clone o repositório:

Abra o seu terminal ou prompt de comando e execute:

```bash
git clone https://github.com/claudineipedrozo/RNLearning.git 

```
2. Navegue até o Diretório do Projeto:

```bash
cd RNLearning 

```
3. Instale as Dependências:

```bash
npm install 

```
4. Executar o Projeto:

```bash
npx expo start
#OU
npm start

```
